ALEXIS


Hello, I’m Alexis, I’m a Law and Language student.
Prior to CodeFirst Girls I didn’t really know anything about coding. I’m graduating this year and wanted to try something new, especially during Lockdown. I’ve always liked exploring and learning languages but never dived into programming languages. After the first week weeks, I was able to see how challenging and rewarding coding can be, similarly to languages I’ve studied.
I’ve really enjoyed learning HTML and CSS.
JavaScript is yet to be my friend, but I’ll definitely work towards it.

REMI


Hey! My name is Remi and I have recently completed my Masters degree in Genetics. At the beginning of this year, I set out a challenge to learn to code (driven primarly by me having failed miserably in a Python module at university). Though I was initially apprehensive about my ability to learn to code again, being a part of Code First: Girls has given me the skills and confidence to be able to tackle code and introduced me to the wonderful world of Web Development. I know I have just touched the tip of the iceberg, but the course's introduction makes me excited for the coding adventures I am yet to embark on.
